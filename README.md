# 📈 US Stock Market Time Series Analysis & Forecasting

This project focuses on the time series analysis and forecasting of the US stock market trends using advanced Deep Learning and Machine Learning models. The goal is to analyze historical market data, extract key macroeconomic indicators, and predict future price movements.

---

## 🚀 Features
* **Data Ingestion:** Automated data collection using financial APIs (e.g., Yahoo Finance).
* **Technical Indicators:** Calculation of Moving Averages (SMA/EMA), RSI, MACD, and Bollinger Bands.
* **Predictive Modeling:** Implementation and comparison of **ARIMA** and Deep Learning **LSTM** (Long Short-Term Memory) networks.
* **Volatility Analysis:** Analysis of market fear index (VIX) and its correlation with major indices.

---

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Data Processing:** Pandas, NumPy
* **Machine Learning & Deep Learning:** Scikit-learn, TensorFlow / Keras
* **Statistical Modeling:** Statsmodels (ARIMA)
* **Visualization:** Matplotlib, Seaborn, Plotly
